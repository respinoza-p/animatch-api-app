# Introduction

Welcome to the documentation for **Croqueton**, a pet adoption matching platform designed to streamline connection management between adopters and rescuers.

The project is structured with a specialized multi-agent developer system on the backend and a reactive Single Page Application (SPA) on the frontend.

## Key Goals
* Provide an interactive questionnaire to evaluate adopters.
* Map behavioral matching algorithms between animals and prospective households.
* Secure and validate all data streams to protect personal records.
* Maintain a highly modular and automated developer workflow.

## Project Directory Map
Here is an overview of the workspace structure:
* `backend/` - The Node.js / Express API server.
* `backup_bd/` - Database backup snapshots.
* `docs/` - GitBook-compatible documentation.
* `.agents/` - Personal guidelines, agent profiles, and skills config.
